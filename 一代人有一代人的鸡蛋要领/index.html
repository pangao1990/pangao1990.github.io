一代人有一代人的鸡蛋要领

本文最后更新于：2026年6月13日 下午

前言

以前领鸡蛋，要排队。现在领“鸡蛋”，要注册。

今天的鸡蛋不是吃的鸡蛋，而是————免费大模型 API Key。

别怕，不写代码也能看懂。三分钟到手，关注我，不迷路。

打开领取入口

先打开 Agnes AI API 平台：

https://platform.agnes-ai.com/login

已有账号，直接登录。没有账号，点击 注册。

注册账号

注册流程很朴素，朴素到像填快递地址：

  • 填写邮箱；
  • 点击验证码输入框右侧的小纸飞机；
  • 去邮箱收验证码；
  • 设置密码；
  • 再确认一次密码；
  • 点击 继续。

恭喜你，已经从“围观群众”进入“领蛋队伍”。

找到鸡蛋领取处

进入后台后，看左侧菜单。

点击：

设置 -> API 密钥

这里，就是本场重点：

鸡蛋领取处。

页面上一般会看到两块：

API 密钥：个人用户用这个。
企业密钥：企业认证用的，先不用管。

我们今天只领个人鸡蛋。

创建新的密钥

在 API 密钥 区域，点击：

创建新的密钥

然后给 Key 起个名字。

填好后，点击 保存。

复制并保存 Key

生成成功后，页面会弹出你的 API Key。

这一步最重要：

先复制。
先保存。
先放好。

API Key 就像家门钥匙。

别发群里。别贴到公开代码仓库。别写进前端页面。别随手截图到处传。

一旦泄露，别人可能拿着你的 Key 去“替你消费”。

检查一下

回到密钥列表，你能看到刚刚创建的 Key。

右侧通常会有：

  • 复制；
  • 编辑；
  • 删除。

看到它，就说明鸡蛋已经稳稳装进篮子里了。

接入文档

官网教程有写怎么接入常见的Agent，比如：OpenClaw，Hermes，Claude，Codex，WorkBuddy等等。

https://agnes-ai.com/doc/cid1

这里需要注意的是，API Base URL 是：

https://apihub.agnes-ai.com/v1

一句话总结

打开平台，注册登录，进入：

设置 -> API 密钥 -> 创建新的密钥 -> 复制保存

免费大模型 API Key，就到手了。

一代人有一代人的鸡蛋要领。

这一代人的要领是：

先拿 Key，再开干。
