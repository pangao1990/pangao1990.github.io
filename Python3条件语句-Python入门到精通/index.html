条件语句

条件语句是通过一条或多条语句的执行结果来决定接下来执行的代码块。

这么理解比较抽象，可以用下面的流程图来展示。

当程序执行到条件语句时，会根据条件语句的执行结果选择不同的路径继续执行程序，没有选择的路径将不会被执行。
