Python 访问网络资源有很多方法，urllib, urllib2, urllib3, httplib, httplib2, requests ，现介绍如下两种方法：

  • 内置的 urllib 模块
    • 优点：自带模块，无需额外下载第三方库
    • 缺点：操作繁琐，缺少高级功能
  • 第三方库 requests
    • 优点：处理URL资源特别方便
    • 缺点：需要下载安装第三方库
