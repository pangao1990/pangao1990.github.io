前言

2021 年最后一天，疫情还没有完全结束，武汉市政府也取消了跨年活动。今晚，就连地铁都提前到 9 点关闭，大家都在家里跨年。这不，我也在家里跨年。不过我并没有看晚会，而是整理了一篇前端小技巧，算是给自己 2021 年一个小小的总结。

正文

用 CSS 隐藏元素有很多种方法，这里介绍 3 种常见的。

opacity: 0

特点是【看不见，占空间，摸得着】

  • 元素隐藏
  • 不改变布局
  • 如果绑定了事件，点击该区域，是可以触发事件的

visibility: hidden

特点是【看不见，占空间，摸不着】

  • 元素隐藏
  • 不改变布局
  • 如果绑定了事件，点击该区域，是无法触发事件的

display: none

特点是【看不见，不占空间，摸不着】

  • 元素隐藏
  • 改变布局
  • 如果绑定了事件，点击该区域，是无法触发事件的
